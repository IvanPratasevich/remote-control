# RSSchool NodeJS Websocket task

## Installation
1. Clone/download repo
2. `npm install`
3. Choose **dev** branch

## Usage
**Development**

`npm run start:dev`

* App served @ `http://localhost:3000` with nodemon

**Production**

`npm run start`

* App served @ `http://localhost:3000` without nodemon
